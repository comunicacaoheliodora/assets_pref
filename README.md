# Design System Prefeitura de Heliodora

Repositorio para centralizar materiais oficiais de identidade visual, componentes, documentos e assets digitais da Prefeitura de Heliodora.

## Estrutura

- `marca/`: arquivos da marca, brasao, logotipos, variacoes e regras de uso.
- `assets/`: imagens, icones, padroes, texturas e arquivos exportados para uso em pecas digitais.
- `design-system/`: tokens, componentes, estilos e especificacoes do sistema visual.
- `modelos/`: templates para apresentacoes, documentos, posts, placas e materiais recorrentes.
- `documentacao/`: guias, instrucoes de uso e decisoes importantes.
- `fontes/`: arquivos de tipografia autorizados para uso institucional, quando aplicavel.

## Como Usar

Baixe ou copie os arquivos necessarios a partir das pastas correspondentes. Sempre preserve os nomes e versoes dos arquivos para evitar uso incorreto de materiais antigos.

## Seguranca

Nunca envie para este repositorio tokens, senhas, chaves de API, arquivos `.env`, certificados, credenciais de acesso ou arquivos internos que nao tenham liberacao para publicacao.

Se algum token ou senha for enviado por engano, remova o arquivo e revogue a credencial imediatamente no servico de origem.

## Organizacao Recomendada

Use nomes claros e consistentes, por exemplo:

```text
logo-prefeitura-heliodora-horizontal-colorido.svg
brasao-prefeitura-heliodora-png-3000px.png
template-post-instagram-campanha-saude.fig
```

Quando um arquivo for atualizado, prefira indicar a versao no nome ou registrar a mudanca na documentacao.
